# My Portfolio Website

This is my personal portfolio website made using React.  
It shows my skills, projects, and contact information.  
The website is fully responsive for both desktop and mobile devices.

## Features
- Home, About, Skills, Projects, and Contact sections  
- Responsive design  
- Simple and clean UI  

## Technologies Used
- React.js  
- HTML  
- CSS / Tailwind CSS  
- JavaScript
